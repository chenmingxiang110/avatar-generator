# avatar-generator
Generate different types of random avatars.

## Requirements

Python3
pytorch>=1.6
cv2>=3.4
tqdm

## 1. Github-like avatars

<img src="imgs/example_githublike.png" width="800" height="100">

```
python generate_githublike.py ./examples/avatars_githublike/ 10 96
```
